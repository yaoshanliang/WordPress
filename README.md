# A framework based on wordpress

## 特点

 * 禁用google字体
 * 关闭版本升级提示
 * 替换Gravatar链接
 * 优化系统打开速度
 * 移除其它无用内容
 
 
## 安装
 
 * 克隆代码
 * 修改根目录wp-config文件为自己的数据库地址、账号、密码
        
        define('DB_NAME', 'ssha');
        define('DB_USER', 'root');
        define('DB_PASSWORD', 'SSHAjlxh8!');
        define('DB_HOST', 'test.sshmt.com');
            
 * 导入根目录wordpress.sql默认数据
